`subdox` - Downloads sublime related documentation.

SYNOPSIS
--------

`subextract` [`-v`|`-h`]  

DESCRIPTION
-----------

`subdox` uses `GTFM` to download sublime related
documentation. See `subdox.ini` for more info.


OPTIONS
-------

`-v`  
Show version and exit.

`-h`  
Show help and exit.

DEPENDENCIES
------------

GTFM
